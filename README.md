# 🚀 Diksha Kailas Mirgal - Personal Portfolio

A sleek, modern, and high-performance personal portfolio built with **React** and **Vite**. This project showcases my journey as a Web Developer, featuring my projects, professional experience, and academic achievements with a premium "Neon Dark" aesthetic.

## ✨ Features

- **Premium UI/UX**: Professional Dark/Neon theme with glassmorphism and smooth transitons.
- **Dynamic Animations**: Scroll-reveal effects and an infinite horizontal marquee for the tech stack.
- **Interactive Experience**: Interactive project cards with direct GitHub/Demo links.
- **Professional Timeline**: A detailed career timeline showcasing internships and roles.
- **Fully Responsive**: Optimized for every screen size—from desktop monitors to mobile devices.
- **Personal Branding**: Integrated high-quality profile imagery and custom typography.

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Modern CSS Variables & Flexbox/Grid)
- **Icons**: SVG & Font icons
- **Deployment Ready**: Modular structure for easy deployment on Vercel, Netlify, or GitHub Pages.

## 📁 Project Structure

```bash
src/
├── components/          # Modular UI components (Hero, About, Projects, etc.)
├── assets/              # Static assets and images
├── App.jsx              # Main layout and section ordering
├── index.css            # Global Design System & CSS Variables
└── main.jsx             # Entry point
```

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/dikshamirgal2005-bit/portfolio.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd portfolio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

5. **Open your browser**:
   Navigate to `http://localhost:5173` to see the site in action.
